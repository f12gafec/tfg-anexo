# tfg-anexo
Anexo del TFG: Predicción de niebla y baja visibilidad atmosférica utilizando técnicas de Machine Learning

Este repositorio contiene el material complementario del Trabajo de Fin de Grado desarrollado por Carolina García Fernández.

# Estructura del Repositorio
* Código: Contiene los scripts de Python y Jupyter Notebooks utilizados para la limpieza de datos, análisis exploratorio y entrenamiento de los modelos de Machine Learning.
* **`/datos`**: Datasets meteorológicos utilizados (o enlace a la fuente oficial si los datos son privados o muy pesados).
* **`/modelos`**: Archivos de los modelos ya entrenados (ej. `.pkl` o `.h5`) listos para su ejecución.
